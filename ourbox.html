Meet OurBox

OurBox is a compact appliance that runs your apps as containers on an internal orchestration layer, and then presents them through a mobile-first web UI that works like a home screen: tap, swipe, press-and-hold—launch apps instantly.

It’s built to be the heart of a home setup that’s:

  • easy enough for non-technical households to use daily
  • powerful enough for builders to extend forever

A home screen for your home server

Most self-hosting dashboards feel like… dashboards. OurBox doesn’t.

The OurBox launcher is a touch-first web UI designed to behave like a phone’s home screen:

  • Swipe through apps
  • Tap to launch
  • Press-and-hold for app actions (open, manage, pin, remove)
  • Add to Home Screen so it opens fullscreen like a real app

Everything is served as standard HTML to your phone—no special client required.

FAQ

Does OurBox require Ethernet?

No—OurBox is designed to work with Wi-Fi. Setup is built to be phone-friendly.

Is it an app or a website?

The launcher is a mobile-first web UI served from the box. You can add it to your phone’s home screen and run it fullscreen like an app.

What kinds of apps can it run?

Containerized services—everything from private home utilities to personal productivity tools. Some apps will be available in an install catalog, and builders can bring their own.

Can I access it when I’m away from home?

Yes—OurBox is designed to support optional remote access through a unique address. Local access remains the default.

Do I need a subscription?

No mandatory subscription. The box is meant to stand on its own. Optional donations help support shared services and long-term maintenance.

Is it for non-technical people or technical people?

Both. The day-to-day experience is designed to be simple. The platform underneath is designed to be extendable.

Is OurBox “privacy magic”?

No product can rewrite reality—but OurBox is built to reduce unnecessary data exposure and make local-first living practical.
